# 部署說明

本專案是免 API Key 的 Streamlit 網頁工具，可放到 GitHub 保存，也可部署到支援 Streamlit 的平台。

## 本機執行

```bash
pip install -r requirements.txt
streamlit run app.py
```

瀏覽器開啟：

```text
http://localhost:8501
```

Windows 使用者也可以直接執行：

```text
一鍵啟動網頁.bat
```

## 上傳 GitHub

上傳前請確認：

- 不要包含真實學生個資。
- 不要上傳 `.streamlit/secrets.toml`。
- 不要上傳 `__pycache__/` 或 log 檔。

專案已提供 `.gitignore`，會自動忽略上述常見不應上傳的檔案。

## Streamlit Cloud 部署

1. 將專案上傳到 GitHub。
2. 到 Streamlit Cloud 建立新 app。
3. Repository 選擇此專案。
4. Main file path 設為 `app.py`。
5. 部署完成後即可使用。

本版本不需要 Secrets 設定。
