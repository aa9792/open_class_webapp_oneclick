教師觀課紀錄表產生器｜免 API Key 本機版

這個版本不需要 OpenAI、Gemini 或任何 API Key。
系統只會讀取你上傳的教案文字與你輸入的基本資料，在本機整理成觀課紀錄表初稿。

第一次使用：
1. 確認電腦已安裝 Python 3.10 以上版本。
2. 執行「一鍵啟動網頁.bat」。
3. 瀏覽器會開啟 http://localhost:8501。
4. 在網頁中上傳 DOCX／PDF／TXT 教案。
5. 輸入基本資料，按「產生觀課紀錄初稿」。
6. 在網頁上修改附表2、附表3、附表4、附表5內容，最後下載 Word。

如果只是要手動填表：
1. 不必上傳教案。
2. 填基本資料後按「建立空白觀課表」。
3. 逐頁填寫附表2至附表5，最後下載 Word。

GitHub 保存提醒：
- 不要上傳含學生個資的真實教案。
- 不要上傳 .streamlit/secrets.toml。
- 本專案已加入 .gitignore，會忽略 secrets、log、快取與虛擬環境。
